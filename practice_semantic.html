Semantic Tags


What are Semantic Tags?

A semantic element clearly describes its meaning to both the browswer and the developer. This is the definition from w3schools. But what I learned is that sematic tags are those which specify a content's value to search engines. It is most important for SEO. Search Engine crawlers would crawl in the webpage to find out what each of the sections inside a webpage contains and learn from it so that it can display it back on Google.


Some semantic tags are stated below:

  • article
  • aside
  • details
  • figcaption
  • figure
  • footer
  • header
  • main
  • mark
  • nav
  • section
  • summary
  • time
